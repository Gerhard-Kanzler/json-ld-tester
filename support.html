What the extension does

JSON-LD & Agent Inspector inspects, validates and tests structured data and agent tools on any website — for technical SEOs, web developers and agencies. Everything runs locally; nothing is uploaded.

  • Detect and validate JSON-LD (plus Microdata & RDFa), with per-block errors and schema.org links.
  • On-page SEO: meta tags, headings, Open Graph & Twitter, hreflang, images, links, word count.
  • HTTP response headers and indexability for the current page.
  • Agent tools (WebMCP) inspection with a safe manual tester and risk classification.
  • Agent readiness score with llms.txt / robots.txt / sitemap / ai.txt checks.

Getting started

  • Open a page that contains structured data (for example, a product or article page).
  • Click the extension’s toolbar icon to open it.
  • Browse the tabs: Structured Data, On-Page, Headers, Agent Tools and Readiness.
  • Use the report button to produce a print-ready inspection report (saved locally as PDF).

Frequently asked

Does it send my data anywhere? No. See the Privacy Policy — nothing is uploaded and there is no tracking.

Why does a tab sometimes show “allow access”? The extension reads a page only on demand and asks for access per site; it does not run in the background on every page you visit.

Which browsers are supported? Safari, Google Chrome (and Chromium browsers) and Mozilla Firefox.
